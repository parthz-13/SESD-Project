# Smart Library & Resource Management System (SLRMS)

## Project Overview
Smart Library & Resource Management System (SLRMS) is a full-stack web application designed to help libraries and educational institutions efficiently manage books, digital resources, members, borrowing workflows, and fines. The system provides role-based access control and supports the complete resource lifecycle from cataloguing to return.

The backend is built using Node.js, Express, and TypeScript, following clean architecture and software engineering best practices. The frontend is built using React and communicates with the backend through RESTful APIs.

## Scope
- User authentication and authorization using JWT
- Role-based access control (Admin, Librarian, Member)
- Book and resource cataloguing and management
- Borrowing, reservation, and return workflows
- Fine calculation and payment tracking
- Notifications for due dates, availability, and overdue items

## Key Features
- Secure login and registration
- Role-based dashboards
- Add, update, and remove books and resources
- Issue and return books to members
- Track borrowing status (Reserved, Issued, Returned, Overdue)
- Automatic fine generation for overdue items
- Notification system for due dates and availability alerts
- REST API-based backend

## Target Users
- Public and institutional libraries
- Educational institutions (schools, colleges, universities)
- Organizations managing internal knowledge/resource bases

## Non-Functional Requirements
- Scalable backend architecture
- Secure and maintainable codebase
- Follows OOP, SOLID, and design pattern principles
- Proper error handling and logging
